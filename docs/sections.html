List taken from packages.debian.org/unstable/ and re-ordered based on my expected usage of them

adminUtilities to administer system resources, manage user accounts, etc.
netDaemons and clients to connect your system to the world.
develDevelopment utilities, compilers, development environments, libraries, etc.
debugPackages providing debugging information for executables and shared libraries.
docFAQs, HOWTOs and other documents trying to explain everything related to Debian, and software needed to browse documentation (man, info, etc).
editorsSoftware to edit files. Programming environments.
textUtilities to format and print text documents.
utilsUtilities for file/disk manipulation, backup and archive tools, system monitoring, input systems, etc.
webWeb servers, browsers, proxies, download tools etc.
databaseDatabase Servers and Clients.
fontsFont packages.
pythonEverything about Python, an interpreted, interactive object oriented language.
rubyEverything about Ruby, an interpreted object oriented language.
shellsCommand shells. Friendly user interfaces for beginners.
graphicsEditors, viewers, converters... Everything to become an artist.
gnomeThe GNOME desktop environment, a powerful, easy to use set of integrated applications.
videoVideo viewers, editors, recording, streaming.
virtualVirtual packages.
httpdWeb servers and their modules.
miscMiscellaneous utilities that didn't fit well anywhere else.
interpretersAll kind of interpreters for interpreted languages. Macro processors.
gamesPrograms to spend a nice time with after all this setting up.
introspectionMachine readable introspection data for use by development tools.
javaEverything about Java.
educationSoftware for learning and teaching.
kernelOperating System Kernels and related modules.
libdevelLibraries necessary for developers to write programs that use them.
libsLibraries to make other programs work. They provide special features to developers.
otherosfsSoftware to run programs compiled for other operating systems, and to use their filesystems.
metapackagesPackages that mainly provide dependencies on other packages.
xfceXfce, a fast and lightweight Desktop Environment.
oldlibsOld versions of libraries, kept for backward compatibility with old applications.
scienceBasic tools for scientific work
soundUtilities to deal with sound: mixers, players, recorders, CD players, etc.
mathMath software.
vcsVersion control systems and related utilities.
x11X servers, libraries, fonts, window managers, terminal emulators and many related applications.
kdeThe K Desktop Environment, a powerful, easy to use set of integrated applications.
zopeZope Application Server and Plone Content Managment System.
mailPrograms to route, read, and compose E-mail messages.
tasksPackages that are used by 'tasksel', a simple interface for users who want to configure their system to perform a specific task.
perlEverything about Perl, an interpreted scripting language.
phpEverything about PHP.
haskellEverything about Haskell.
texThe famous typesetting software and related programs.
localizationLocalization support for big software packages.
gnu-rEverything about GNU R, a statistical computation and graphics system.
gnustepThe GNUstep environment.
ocamlEverything about OCaml, an ML language implementation.
lispEverything about Lisp.
electronicsElectronics utilities.
embeddedSoftware suitable for use in embedded applications.
debian-installerSpecial packages for building customized debian-installer variants. Do not install them on a normal system!
cli-monoEverything about Mono and the Common Language Infrastructure.
commSoftware to use your modem in the old fashioned style.
hamradioSoftware for ham radio.
newsSoftware to access Usenet, to set up news servers, etc.
